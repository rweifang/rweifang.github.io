Note-OS-8.4 Method for Handling Deadlocks

發表於 2019-11-10 分類於 Operating System Disqus：

Method for Handling Deadlocks

大致上來說，處理deadlock的方法有以下幾種：

  • 忽視問題，假裝deadlock沒有發生在系統中。
  • 定出一個協議，確保系統不會進入deadlock state。
  • 允許系統進入deadlock state，但是去偵測它、修正它。
